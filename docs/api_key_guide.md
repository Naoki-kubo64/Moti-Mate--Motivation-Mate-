## ğŸ‡ºğŸ‡¸ English Guide

1. **Access Google AI Studio**

   - Go to [Google AI Studio (Get API Key)](https://aistudio.google.com/app/apikey).
   - Sign in with your Google Account.

2. **Create API Key**

   - Click the **"Create API Key"** button.
   - You can choose to create a key in a new project (recommended) or an existing Google Cloud project.
   - Click **"Create API key in new project"**.

3. **Copy the Key**

   - After a few seconds, your API Key (starting with `AIzaSy...`) will be displayed.
   - **Copy** this key. (Do not share it with others!)

4. **Paste into Motivation Mate**
   - Open **Settings** in Motivation Mate.
   - Go to the **"Intelligence"** tab.
   - Paste the key into the **"Gemini API Key"** field.
   - Click **"Save Key"**.
