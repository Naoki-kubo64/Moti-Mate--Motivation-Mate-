# Motivation Mate (Moti-Mate)

Motivation Mate は、あなたの PC 画面を監視し、設定した目標に向かって作業しているかを見守る「モチベーション管理ウィジェット」です。
AI (Google Gemini Pro Vision) があなたのサボりを検知し、励ましたり叱ったりしてくれます。

## 機能

- **画面監視**: バックグラウンドで画面をチェック。
- **AI フィードバック**: Gemini Pro Vision が画面内容を分析し、状況に応じたコメントを表示。
- **ウィジェット**: 邪魔にならないコンパクトなサイズ。

## 必要なもの

- Python 3.10+
- Google Cloud Project で取得した **Gemini API Key**

## インストール

1. リポジトリをクローンまたはダウンロードします。
2. 必要なライブラリをインストールします。

```bash
pip install -r requirements.txt
```

## 使い方

1. アプリを起動します。

```bash
python main.py
```

2. **Gemini API Key** を入力します。
3. **現在のモード（目標）** を入力します。（例：「プログラミング」「資格の勉強」など）
4. **フィードバック間隔** を調整します。
5. 「監視開始」ボタンを押します。
6. 設定した間隔ごとに AI からメッセージが届きます。頑張りましょう！

## 注意事項

- スクリーンショットは API 送信のために一時的にメモリに保存されますが、ディスクには保存されません。
- API 送信後、画像データは速やかに破棄されます。
